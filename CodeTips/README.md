# vscode tips
- 对python而言，run和debug下sys.path为运行当前文件所在目录+默认指定的目录。
- os.path.abspath('.‘)为终端pwd,vscode运行python代码时，会进入到当前python文件所在路径
```sys.path.append(os.path.abspath('.'))```
